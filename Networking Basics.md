***Three way Handshake*** 
	- Client Sends SYN packet.
	- Server Responds to it with a SYN-ACK packet.
	- Client sends back an ACK packet to initiate connection.


***OSI Model (Please Do Not Throw Pizza Sausage Away)***
	- Physical Layer: Ethernet, cables.
	- Data Link Layer: Switches, MAC addresses.
	- Network Layer: IP addresses, routers, routing.
	- Transport Layer: TCP, UDP protcol.
	- Session Layer: Session management.
	- Presentation Layer: SSL/TLS, jpeg.
	- Application Layer: HTTP, HTTPS, SMTP.
